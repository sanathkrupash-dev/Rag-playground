# Prompt Engineering Library — AI PM Portfolio

This library contains reusable prompt patterns used in the RAG Playground and agent workflows.

---

## 1. Instructional Prompt (SOP-style)
“You are a technical documentation assistant. Rewrite the following process into a simple 5-step SOP…”

---

## 2. Data Extraction Prompt
“Extract entities from the following text:
- Roles
- Snowflake resources
- Data flows
Return JSON only.”

---

## 3. Query Rewrite Prompt
“You are a semantic search optimizer. Rewrite the following user query into 3 search-friendly variations.”

---

## 4. Error Explanation Prompt
“You are a debugging assistant. Explain the error in simple terms and provide a fix.”

---

## 5. Email/Slack Rewrite Prompt
“You rewrite text to be concise, clear, and executive-friendly.”

---

## 6. PM-Style Summary Prompt
“Summarize this into:  
- Context  
- Problem  
- Impact  
- Recommendation”
